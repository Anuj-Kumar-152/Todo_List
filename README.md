# React Todo Manager

-> A simple and responsive Todo application built using React, Vite, and Tailwind CSS. It allows users to manage daily tasks efficiently.

# Features 
   - Add new tasks  
   - Edit existing tasks 
   - Delete tasks 
   - Mark tasks as completed 
   - Completed tasks highlighted visually 
   - Data persists after page refresh using localStorage 
   - Fully responsive (mobile, tablet, desktop)

# Tech Stack 
   - React – for building UI 
   - Vite – development environment 
   - Tailwind CSS – styling 
   - JavaScript – application logic 
   - localStorage – data persistence

# Installation 
   Clone the repository: git remote add origin https://github.com/Anuj-Kumar-152/Todo_List.git 

   - Go to the project folder: cd Todo_List
 
   - Install dependencies: 
   - npm install  
   - Run the project: 
   - npm run dev

# Project Structure
src
 ├─ App.jsx          // Main application logic
 ├─ components       // Reusable UI components
 ├─ hooks            // Custom hooks (localStorage logic)

# Description

This project demonstrates how to build a real-world React application using component-based architecture and persistent state management. Todos are stored in localStorage so they remain available even after refreshing the page.
